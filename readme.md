## Preguntas Analiticas

# Ejercicio 1 - 
Crear una función constructora nos ayuda mucho porque no tenemos que escribir el mismo objeto muchas veces. Solo defino una vez la estructura del computador y después puedo crear varios(pcs) con datos distintos, sin repetir código.
# Ejercicio 2
Usar this dentro del método sirve porque hace que el objeto se refiera a sus propias propiedades. Así el método puede leer la información correcta de esa mascota específica y no de otra.

# Ejercicio 3
Que el objeto conozca su propio estado hace que el código esté más ordenado y tenga más lógica.En vez de calcular si aprobó por fuera, el mismo estudiante sabe si aprobó o no según la nota que saco.

# Ejercicio 4
Si se no controlara bien el estado del libro, podrían pasar errores como prestar un libro que ya está prestado o devolver uno que nunca salió. Por eso es importante validar antes de cambiar el estado.

# Ejercicio 5
Dejar que el usuario ingrese los datos hace que el programa sea más flexible y real. Así no se depende de valores escritos a mano en el código, sino que puedo registrar vehículos distintos cada vez que se ejecute.
